# RosterManagementSystem

## Description
A system to manage and update student rosters by reading and writing student-class assignments from a file.

## Features 
- Reads roster data from text file.
- Processes and updates roster information.
- Writes back changes to the roster file.

## How It Works
1. The program reads the existings rosters from a text file.
2. The data is stroed and processed as needed.
3. Update roster is written back to the text file. 

## Purpose
Project was created to practice file input/output and data management concepts.
